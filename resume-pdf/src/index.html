Profil

Les possibilités de connexions et d’interfaces qu’offre le web sont innombrables. Téléphone, tablette, tv, montre, capteur, … les supports se multiplient depuis ces dernières années, et de nombreux projets imaginent de nouvelles manières d’échanger. Ce sont ces idées qui m’intéressent dans le web.

Mon domaine d’expertise est le développement, design et code, mais je m’intéresse à tous les aspects d’un projet. J’ai été développeur, chef de projet, responsable d’équipe, responsable avant vente, auto entrepreneur, designer dans différents contextes et projets, j’attends la prochaine aventure avec impatience.
